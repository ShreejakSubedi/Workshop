Flexbox vs Grid

Flexbox is used for one-dimensional layouts (row OR column), while Grid is used for two-dimensional layouts (rows AND columns). Flexbox is best for components and alignment, while Grid is best for page layout structure. In this website, Flexbox is used for layout alignment such as navigation, hero section, and projects layout, while Grid is used for the skills section and gallery layout.
